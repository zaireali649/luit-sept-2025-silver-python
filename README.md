# luit-sept-2025-silver-python

This repository is for students learning basic Python for the first time.  

## Repository Structure

```
.
├── .gitignore        # Standard Git ignore file to exclude unnecessary files
├── hello_world.py    # First Python script with a simple "Hello, World!" example
└── README.md         # Project documentation
```

## Getting Started

1. **Clone the Repository**  
   Open your terminal and run:
   ```bash
   git clone <your-repo-url>
   cd luit-sept-2025-silver-python
   ```

2. **Run the Hello World Program**  
   Make sure you have Python 3 installed. Then run:
   ```bash
   python hello_world.py
   ```

   You should see the following output:
   ```
   Hello, World!
   ```

## Purpose

This repo serves as your starting point to practice Python. Right now it contains a single file, `hello_world.py`, which prints a message to the screen. This is often the first program written when learning any programming language.
